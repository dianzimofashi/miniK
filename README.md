
### ✅ 依赖项

| 组件                  | 版本要求             | 说明                         |
|-----------------------|----------------------|------------------------------|
| Visual Studio         | 2012 及以上          | 推荐使用 VS2012 或 VS2015   |
| .NET Framework        | 4.5 及以上           | 基于 WinForms 构建           |
| ArcGIS Engine Runtime | 10.2.2               | 必须正确安装并授权           |
| ArcObjects SDK        | 10.2.2               | 用于调用 GIS 核心接口        |
| ArcSDE                | 10.2.2               | 可选，用于连接企业级数据库图层 |
